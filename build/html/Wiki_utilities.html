Wiki_utilities module

Created on Wed May 4 14:25:07 2022

@author: 3b13j

Contains some functions that help handling phonetically transcribed words available on wikipedia

Wiki_utilities.dic2word(dic)[source]

converts a wiki dic into a dic of "word" objects

Wiki_utilities.get_language(path, name)[source]

builds a Language object from data found on wiktionnary Final step of the conversion that reunites all the previous functions.

pathstr

path to a file extracted from wiktionnary.

lg : Language

Wiki_utilities.segm2syl(dic)[source]

take as input the dictionnary created out of the wiki data and used the info concerning syllabification to create syllables in our IPA format

Wiki_utilities.treat_syl(syl, stress=False)[source]

Converts a syllable in the syllabled jey of the vocabulary into a syllable object. Effectuates the actual encoding of data into the objects used in this program.

syl : str stress : TYPE, optional

DESCRIPTION. The default is False.

syl : a Syllable object

Wiki_utilities.wiki_lexicon(path)[source]

extracts the information from the iki files

pathstr

path to the file we read

dico

keys : simple orthograph of the word. values : the IPA and syllabled representation of the word give by the wiktionnary websiote.
