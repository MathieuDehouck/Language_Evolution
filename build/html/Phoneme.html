Semantic of a feature ;

syl : field of the Phoneme voice : filed of the Phoneme

Features :
First tuple :

0 : "place of articulation" : int 1 : list of 5 manner of articulation , each coded by a boolean

"plosive" "fricative" "nasal" "trill" "lateral"

Second tuple :

"secondary place of articulation" int (3 or 4 possibilities, same semantics as in place of articulation) "pren_nasal" bool "aspiration" bool
